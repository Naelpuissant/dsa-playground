# Usefull Datastructs

Usefull datastructs is a datastructs playground for real life usecases.

* [X] ordered linked list : just to warm up for the skiplist
* [X] skiplist (sl) : a simple skiplist inspired by [huandu/skiplist](github.com/huandu/skiplist) and [sean-public/fast-skiplist](https://github.com/sean-public/fast-skiplist/) but for my needs
* [ ] Arena based skiplist 
* [X] BitMap : a bitmap is necessary to achieve a bloom filter without wasting memory
* [X] Bloom filter : use my bit map (please provide a fast, non-cryptographic hash function such as murmur3)
